# myblog

项目名字是 blog  下载后要把项目名称改成blog

以前做过的blog，现在来整理一下

后端使用的技术框架是SpringBoot+jpa+Thymeleaf

数据库mysql

前端UI：Segment UI组件

工具：IDEA Maven JDK8
